setSendLaunchCrashesSynchronously

open fun setSendLaunchCrashesSynchronously(sendLaunchCrashesSynchronously: Boolean): Unit

Sets whether or not Bugsnag should send crashes synchronously that occurred during the application's launch period. By default this behavior is enabled. See #setLaunchDurationMillis(long)
